Detailed Description

template<typename T>
class WECore::MONSTR::MONSTRCrossover< T >

Class which provides three band stereo width control.

This class contains three MONSTRBand objects, and may later be expanded to support a variable number of bands.

To use this class, use the setter and getter methods to manipulate crossover parameters which are shared by several bands (crossover frequencies), and call the setter and getter methods of each band to manipulate parameters which are particular to each band. Call the reset, setSampleRate and process methods as required.

See Also
MONSTRBand

The below example shows how to prepare a MONSTRCrossover object to process audio:

MONSTRCrossover crossover;
crossover.setSampleRate(44100);

Then when you have a buffer ready:

crossover.Process2in2out(leftSample, rightSample, numSamples);
